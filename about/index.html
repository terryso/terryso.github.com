About the author

Donovan is a Dublin based front end web developer who enjoys making web sites and projects. He has been building for the web for close to 15 years, during which time he has worked both on the server-side development as well as the front-end design of a wide range of projects.

Having put in a fair amount of time building PHP and MySQL based websites, and managing Ubuntu server back-ends, Donovan spends more time nowadays working with front-end CSS, HTML and Javascript. Donovan also builds and runs several personal websites, including Shop Ireland, and is employed as the Director of UX for mobile CRM company, Converser.

Donovan has also given several talks, including an introduction to AngularJS, CSS3 Animation and an introduction to Node.js.

If you'd like to get in touch, please use d@hop.ie, @donovanh on Twitter, or LinkedIn.
